Providers represent platforms which Krok supports. I.e.: Github, Gitlab, Gitea, Bitbucket, etc. VCS providers.

To get a list of supported platforms run:

```
./bin/darwin/amd64/krokctl list platforms
ID      NAME
1       github
2       gitlab
3       gitea
```
